## Tagline
Security scanner for OpenClaw skills - scan before you install

## Description
OpenClaw Verifier protects you from malicious OpenClaw skills. With 32.6% of ClawHub skills marked as CRITICAL, you need to verify before installing.

## Features (bullets)
- 🔍 Detects malicious code patterns
- 🎯 5 security categories: eval/exec, external API, file access, network, crypto keys
- 📊 Security score 0-100 with risk level
- 🖥️ CLI + Web UI
- 🔄 GitHub Actions integration
- 💻 Open source (MIT)

## Maker Comment
Hi Product Hunt! 👋

I built OpenClaw Verifier after discovering that 32.6% of ClawHub skills contain critical security issues, and 341 malicious skills were found in the ClawHavoc attack.

OpenClaw is an amazing AI agent framework with 180K+ GitHub stars, but the skill ecosystem has a security problem. Users were installing skills blindly, exposing their API keys and data.

OpenClaw Verifier solves this by:
- Scanning skills before installation
- Detecting malicious patterns
- Providing a security score
- Integrating with CI/CD

It's open source, free to use, and works both as a CLI tool and Web UI.

Would love your feedback! 🦞
